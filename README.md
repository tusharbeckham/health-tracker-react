# Health Tracker 🏥

A modern, cross-platform health tracking application built with **React + Vite** and powered by **Capacitor** for a native mobile experience.

This application helps users track their daily health and fitness activities, build consistent habits, and achieve their wellness goals with an intuitive, clean, and "Native-feel" interface.

## 📱 Mobile Support (New!)
The project now includes **Capacitor integration**, allowing it to run as a native Android application. 
- **APK Download:** You can find the latest build in the [Releases](../../releases) section.
- **UI Optimization:** Fixed blue-highlight tap issues and enabled native touch interactions.

## 🚀 Key Features
- **Health Dashboard:** Personalized insights and daily summaries.
- **Step Tracking:** Monitor daily goals with visual progress bars.
- **Water Intake:** Logging and monitoring daily hydration.
- **Workout Logs:** Detailed exercise tracking.
- **Streak System:** Built-in daily and weekly streak counters to keep you motivated.

## 🛠️ Tech Stack
- **Frontend:** React + Vite
- **Mobile Bridge:** Capacitor (Android)
- **Styling:** Modern CSS3 with Responsive Design
- **Build Tool:** Vite for lightning-fast HMR

## 🏗️ Getting Started

### For Web:
1. Clone the repository: `git clone <repo-url>`
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`

### For Android:
1. Build the web project: `npm run build`
2. Sync with Android folder: `npx cap sync android`
3. Open in Android Studio: `npx cap open android`
